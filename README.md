⚔️ TopDownGame
Shadow of the Skeleton: Dungeon Crusade — это динамичная 2D top-down игра, в которой игроку предстоит пробиться через орды скелетов в одной мрачной локации, используя меч и молниеносный рывок. Окружение не просто декорация — часть препятствий можно сокрушить ударом, открывая новые пути к финальной битве с могущественным Боссом-Скелетом.

🎮 Геймплей
Игра выполнена в жанре top-down 

🛡️ Атака мечом: Основной способ нанесения урона. Меч может не только уничтожать врагов, но и разрушать определенные препятствия на карте (камни, кусты).

💨 Рывок (Dash): Маневр уклонения и быстрого перемещения. Позволяет мгновенно уходить от атак скелетов или сокращать дистанцию для удара. 

🏛️ Разрушаемое окружение: Часть интерьера на карте можно разрушить с помощью удара мечем.

🧟 Враги:

Скелеты: Обычные противники, патрулирующие локацию. Атакуют при приближении игрока.

Босс-Скелет: Финальный враг. Обладает увеличенным запасом здоровья, скоростью и размерами.

🗺️ Локация
В игре представлена одна цельная локация.

🕹️ Управление
Действие	Клавиша
Перемещение	W, A, S, D ИЛИ стрелки
Атака мечом	Левая кнопка мыши
Рывок (Dash)	Пробе

Проект разработан с использованием:
